## 1. 사이트 접속

### 접속 URL
**http://remo-data-bridge.remo.re.kr:3015/**

### 접속 방법
1. 웹 브라우저(Chrome, Edge, Safari 등)를 실행합니다.
2. 주소창에 위 URL을 입력합니다.
3. Enter 키를 눌러 사이트에 접속합니다.

### 권장 브라우저
- Google Chrome (최신 버전)
- Microsoft Edge (최신 버전)
- Safari (최신 버전)

## 2. 로그인

![로그인 페이지](./screenshots/01_login.png)

### 설명
- 시스템에 접속하기 위한 로그인 페이지입니다.
- 이메일과 비밀번호를 입력하여 로그인합니다.

### 사용 방법
1. 이메일 주소를 입력합니다.
2. 비밀번호를 입력합니다.
3. **로그인** 버튼을 클릭합니다.

### 계정 생성
- 계정이 없는 경우, REMO에 계정 생성을 요청해야 합니다.
- 계정 생성 요청: **info@remo.re.kr**

## 5. 새 요청 생성

### 5-1. 개인정보 이전 동의

![동의 팝업](./screenshots/04a_requests_new_consent.png)

### 설명
- 새 요청 생성 시 **개인정보 국외 이전 동의** 팝업이 먼저 표시됩니다.
- 데이터가 분석을 위해 해외 서버로 전송되므로 사용자 동의가 필요합니다.

### 사용 방법
1. 동의 내용을 확인합니다.
2. 하단의 동의 체크박스를 클릭합니다.
3. **동의 및 진행** 버튼을 클릭합니다.
4. 동의하지 않으려면 **취소** 버튼을 클릭하여 이전 페이지로 돌아갑니다.

### 5-2. 요청 정보 입력

![새 요청 폼](./screenshots/04b_requests_new_form.png)

### 설명
- 동의 후 실제 분석 요청을 작성하는 폼입니다.

### 사용 방법
1. **요청 제목**을 입력합니다.
2. 분석에 필요한 **파일을 업로드**합니다.
3. 추가 설명이 있다면 **메모**를 작성합니다.
4. **요청 생성** 버튼을 클릭합니다.

### 파일 업로드
- 드래그 앤 드롭으로 파일을 업로드할 수 있습니다.
- 여러 파일을 동시에 업로드할 수 있습니다.
- 업로드된 파일은 AES-256 암호화로 안전하게 전송됩니다.

## 6. 요청 상세

![요청 상세](./screenshots/05_request_detail.png)

### 설명
- 개별 분석 요청의 상세 정보를 확인하는 페이지입니다.

### 주요 기능
- 요청 정보 확인 (제목, 상태, 생성일 등)
- 업로드된 파일 목록 및 다운로드
- 분석 결과 확인
- 결과 파일 다운로드

### 파일 다운로드
- 파일명을 클릭하면 해당 파일을 다운로드할 수 있습니다.
