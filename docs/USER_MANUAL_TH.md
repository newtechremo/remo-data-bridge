## 1. การเข้าถึงเว็บไซต์

### URL สำหรับเข้าถึง
**http://remo-data-bridge.remo.re.kr:3015/**

### วิธีเข้าถึง
1. เปิดเว็บเบราว์เซอร์ (Chrome, Edge, Safari ฯลฯ)
2. ป้อน URL ด้านบนในแถบที่อยู่

### เบราว์เซอร์ที่แนะนำ

## 5. สร้างคำขอใหม่

### 5-1. ความยินยอมในการโอนย้ายข้อมูล

![ป๊อปอัพความยินยอม](./screenshots/04a_requests_new_consent.png)

### คำอธิบาย
- เมื่อสร้างคำขอใหม่ จะแสดงป๊อปอัพ **ความยินยอมในการโอนย้ายข้อมูลข้ามพรมแดน** ก่อน
- จำเป็นต้องได้รับความยินยอมจากผู้ใช้เนื่องจากข้อมูลจะถูกโอนไปยังเซิร์ฟเวอร์ต่างประเทศเพื่อการวิเคราะห์

### วิธีใช้งาน
1. ตรวจสอบข้อมูลความยินยอม
2. คลิกช่องทำเครื่องหมายความยินยอมด้านล่าง
3. คลิกปุ่ม **ยอมรับและดำเนินการต่อ**
4. หากต้องการปฏิเสธ คลิกปุ่ม **ยกเลิก** เพื่อกลับไปหน้าก่อนหน้า

### 5-2. กรอกข้อมูลคำขอ

![แบบฟอร์มคำขอใหม่](./screenshots/04b_requests_new_form.png)

### คำอธิบาย
- นี่คือแบบฟอร์มสำหรับสร้างคำขอวิเคราะห์หลังจากให้ความยินยอม

### วิธีใช้งาน
1. กรอก **หัวข้อคำขอ**
2. **อัปโหลดไฟล์** ที่จำเป็นสำหรับการวิเคราะห์
3. เพิ่ม **บันทึก** หากมีหมายเหตุเพิ่มเติม
4. คลิกปุ่ม **ส่งคำขอ**

### การอัปโหลดไฟล์
- คุณสามารถอัปโหลดไฟล์โดยการลากและวาง
- สามารถอัปโหลดหลายไฟล์พร้อมกันได้
- ไฟล์ที่อัปโหลดจะถูกโอนอย่างปลอดภัยด้วยการเข้ารหัส AES-256

## 6. รายละเอียดคำขอ

![รายละเอียดคำขอ](./screenshots/05_request_detail.png)

### คำอธิบาย
- หน้านี้แสดงข้อมูลรายละเอียดเกี่ยวกับคำขอวิเคราะห์แต่ละรายการ

### ฟีเจอร์หลัก
- ดูข้อมูลคำขอ (หัวข้อ, สถานะ, วันที่สร้าง ฯลฯ)
- ดูรายการไฟล์ที่อัปโหลดและดาวน์โหลด
- ตรวจสอบผลการวิเคราะห์
- ดาวน์โหลดไฟล์ผลลัพธ์

### การดาวน์โหลดไฟล์
- คลิกที่ชื่อไฟล์เพื่อดาวน์โหลดไฟล์
